= 根据已提交容量计费
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
承诺容量是购买订阅时为特定性能服务级别承诺的容量。

承诺容量可以是您和 NetApp/合作伙伴接受的单个订阅中各种性能服务级别的总容量。此容量会在每个Keystone订单上进行说明、并进行计费、而不管实际容量消耗如何。

有关 Keystone 支持的不同容量的信息，请参阅link:../concepts/supported-storage-capacity-v2.html["Keystone支持的存储容量"] 。
