= 获取Keystone帮助
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
NetApp 为 Keystone 客户提供运营服务。客户可通过 ITOM 解决方案获得专用的 Keystone Success Manager (KSM)、直接支持联系选项和主动监控。



== NetApp 支持服务

NetApp 为 Keystone 客户提供远程运营服务。这些服务涵盖一系列存储管理活动，包括资产和配置管理、容量和性能管理、变更管理、事件和事故管理、服务请求履行和报告。

.Keystone Success Manager
您的 KSM 在 Keystone 服务方面与您密切合作，并提供每周或每月的账单和运营报告。具体职责在您的 NetApp Keystone 协议中定义。



== 联系支持部门

如果您的 Keystone 订阅遇到问题，请使用您的 NSS 凭据登录到 link:https://support.netapp.com["NetApp 支持站点"^] 以创建支持案例。

如果您没有 NSS 登录，link:https://mysupport.netapp.com/site/user/registration["注册您的帐户"^]请使用您订阅中的一个节点的序列号 (SN)。

有关屏幕截图的分步说明，请参见 link:https://kb.netapp.com/nss/Support_Site/How_to_open_a_technical_support_case["如何创建技术支持案例"^]。

如果您希望直接与支持代表通话，请拨打以下其中一个电话号码：

* 美国/加拿大：888 463 8277
* 欧洲：00 800 44 638277
* 印度：000 800 100 8948
* 更多全球数字：link:https://www.netapp.com/company/contact-us/support/["https://www.netapp.com/company/contact-us/support/"^]




== 支持监控

NetApp 使用 ITOM 监控解决方案来主动监控产品和订阅服务的运行状况，并连接到 NetApp Keystone 环境进行故障排除。
