= 了解由Equin.提供支持的主机代管服务
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
为了提供全堆栈解决方案即服务、NetApp与Equinx建立了合作伙伴关系、在Equinx数据中心托管NetApp Keystone STaaS服务。

由Equynox提供支持的Keystone主机代管(Co-Lo)服务与标准Keystone服务产品保持不变。

在此服务中：

* Equ无限 空间、电耗、散热、网络、发票、 和存储。
* 为某些Keystone销售申请提供支持。
* 这些服务在11个国家/地区的数据中心内均受支持。
