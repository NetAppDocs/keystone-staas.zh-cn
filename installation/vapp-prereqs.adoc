= 虚拟基础架构要求
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
在VMware vSphere系统上安装Keystone收集器需要一些虚拟基础架构配置。

Keystone收集器服务器虚拟机应具有以下内容：

* 操作系统：VMware vCenter Server和ESXi 6.5或更高版本
* 核心：4个CPU
* RAM：12 GB RAM
* 磁盘空间：200 GB vDisk
