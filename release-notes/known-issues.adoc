= 已知问题
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs:

[role="lead"]
已知问题用于确定可能会妨碍您有效使用Keystone订阅服务的问题。

NetApp Keystone STaaS中报告了以下已知问题：

[cols="3*"]
|===
| 已知问题描述 | 说明 | 临时解决策 


 a| 
显示的QoS策略类型不正确
 a| 
在*卷详细信息*选项卡中，如果未应用QoS策略，*QoS策略类型*列将显示_QoS_，而符合列将显示_not set_，从而导致显示的QoS策略状态不一致。
 a| 
无



 a| 
主订阅和链接订阅不提供卷详细信息
 a| 
"*卷摘要*"选项卡中的主订阅和链接二级订阅的卷总数、QoS合规性状态、受保护卷计数以及总已用容量均显示为零。
 a| 
无

|===
