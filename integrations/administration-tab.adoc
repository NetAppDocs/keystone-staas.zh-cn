= 查看 Keystone 订阅的服务请求
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
您可以通过 BlueXP 查看和跟踪修改 Keystone 订阅的服务请求，提供快速摘要并监控进度。

要查看进度或取消请求，请按照以下步骤操作：

.步骤
. 从 BlueXP 左侧导航菜单中，选择 *存储 > Keystone > 管理*。
+
您会看到所有服务请求的列表，包括请求编号、请求类型和当前状态。

. 您可以点击image:down-arrow.png["向下箭头图标"]点击“提交日期”列旁边的图标，展开请求编号并查看详细信息。您可以看到提出请求的性能服务级别。
+
image:bxp-service-request-list.png["屏幕截图显示了针对哪些性能服务级别提出了修改请求"]

. 选择请求编号可查看详细信息，包括所请求订阅的修改详细信息和当前进度状态。
+
image:bxp-service-progress.png["显示当前进度状态的屏幕截图"]
